This program implements a sequentialized version of P-SCAN-3 in clrs, for an array of cummulative sum. It takes an input file in the format:
. First line contains a number C>= 0 of test cases n the file.
. Then comes C test cases, each of the following form:
	- Line containing the number n>=0 of values in the test.
	- Line containing the n values with exactly one space separation between each number.
